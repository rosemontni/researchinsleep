# Gleason Theorem Accessible Project

This example studies:

- [Cooke--Keane--Moran, "An elementary proof of Gleason's theorem"](https://rogermcooke.net/rogermcooke_files/Elementary%20Proof%20of%20Gleasons%20Theorem.pdf)

Project goals:

1. understand the Cooke proof deeply
2. reconstruct as much of it locally as possible
3. separate what is fully rebuilt from what is still imported
4. search for a more accessible explanation path that avoids physics-first
   presentation
