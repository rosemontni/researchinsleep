# Submission Package

Prepare a separate package for each finalist under `submissions/<journal-slug>/`.

## Minimum Package Contents

- `README.md`
  - what is included
  - manuscript version
  - journal name
  - date prepared
- `cover-letter.md`
  - concise venue-specific cover letter draft
- `journal-fit.md`
  - why this venue is suitable
  - why it is reachable
  - known risks
- `style-notes.md`
  - writing and presentation conventions learned from the 20-paper sample
- manuscript source
  - adapted source files or a clearly named copy of the paper folder
- `submission-checklist.md`
  - author guideline requirements
  - formatting status
  - unresolved unknowns

## Cover Letter Guidance

Include:

- manuscript title
- concise contribution summary
- why the journal's audience is a good match
- a restrained statement of significance

Do not exaggerate novelty or impact beyond the evidence.

## Style Notes Guidance

Summarize:

- how titles are framed
- what abstracts emphasize
- how introductions motivate the work
- how results are narrated
- how discussion and limitations are handled
- common figure density and table style

## Submission Checklist Guidance

Record:

- manuscript type
- page or word constraints
- abstract limits
- figure limits if any
- supplementary requirements
- data or code availability expectations
- ethics, conflict, funding, or disclosure statements
- cover letter requirements

If any requirement is unknown, write `UNKNOWN` instead of guessing.
